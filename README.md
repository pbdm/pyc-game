# Base Defense (保护基地)

A tower defense game built with Phaser 3 and Vite.

## How to Play

1. **Install Dependencies**: `npm install`
2. **Start Dev Server**: `npm run dev`
3. **Build for Production**: `npm run build`

## Controls

*   **Left Click**: Select items from the shop (bottom panel) and place them on the grid.
*   **Categories**: Click on Turret/Trap/Defense/Tool text to switch shop categories.
*   **Start Wave**: Click the "Start Wave" button to spawn zombies.

## Game Goal

Defend the base (Blue Square on the right) from 21 waves of zombies. Place turrets and traps to defeat them.
